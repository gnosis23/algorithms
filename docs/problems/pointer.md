### 不定长滑动窗口

求最长

- [LC3. 无重复字符的最长子串](): 缓慢移动右侧，遇到重复移动左侧直到不重复
- [LC3090. 每个字符最多出现两次的最长子字符串]()
- [LC1493. 删除一个元素后全为 1 的最长子数组]()
  - 注意要求必须删掉一个（肯定优先删除 0，否则删除 1）
  - 滑动窗口，碰到第一个 0 删除
- [LC3634. 使数组平衡的最少移除数量](): 最大值/最小值不能超过 k
  - 先排序

求最短

- [LC209. 长度最小的子数组](): 求目标 target 的最小子数组长度
  - 缓慢移动右侧，如果和大于 target ，重复移动左边并更新答案
- [LC3795. 不同元素和至少为 K 的最短数组长度](): 不能包含重复数组
  - 使用 map 来维护数字数量，出去的时候-1，进来+1
